# ShriSSSSanstha.com

A complete modern NGO website for ShriSSSSanstha, built with React.js, Bootstrap 5, custom CSS, AOS animations, carousel testimonials, controlled forms, donation interactions, and responsive layouts.

## Run Locally

```bash
npm install
npm start
```

The app runs on Vite, usually at `http://localhost:5173`.

## Tech Stack

- React.js
- Bootstrap 5 and React Bootstrap-ready dependencies
- AOS scroll animations
- React CountUp and Intersection Observer
- React Slick carousel
- React Icons
- React Scroll
- Framer Motion dependency included

## Sections

- Loader
- Sticky Navbar
- Hero with impact stats
- About with animated counters
- Programs and objectives
- Gallery
- Donation form
- Testimonials carousel
- Volunteer form
- Blog cards
- Contact and map
